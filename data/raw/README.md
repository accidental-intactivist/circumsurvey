# This directory is for your raw survey CSV export

Place your Google Forms CSV export here as `responses.csv`.

**⚠️ This directory is gitignored.** The raw CSV never leaves your local machine
and is never committed to the repository. Only pre-computed aggregates
(output of the scripts in `/scripts/`) are committed and deployed.

## File to place here:
- `responses.csv` — Export from Google Forms (The Accidental Intactivist's Inquiry)
